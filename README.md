## Grading

| Criterion | Points | What we evaluate |
|-----------|:------:|------------------|
| **Product** | 2-5 | Works, stable, has features |
| **Users** | 2-5 | Number of active users |
| **Marketing** | 2-5 | Landing, content, visibility |
| **Sales** | 2-5 | Revenue or conversion |
| **Growth** | 2-5 | Metrics trend over semester |
